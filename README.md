# Find a Fundi

Find a Fundi is a C2C e-commerce web application that connects clients with local South African service providers.

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## User Roles

- Administrator
- Fundi / Service Provider
- Client

## Main Features

- User registration and login
- Role-based dashboards
- Service listing management
- Service browsing
- Booking system
- Simulated payment process
- Reviews
- Admin user and service management
